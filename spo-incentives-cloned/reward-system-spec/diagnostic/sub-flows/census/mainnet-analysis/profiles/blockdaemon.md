## Identity

Blockdaemon is an enterprise blockchain infrastructure provider — "institutional gateway to Web3." Products: staking (app + APIs), MPC wallets & vaults, dedicated nodes, REST/RPC APIs. Supports 70+ PoS networks for staking, 50+ for node infrastructure. ISO 27001 certified, SOC 2 Type II compliant. Claims $110B+ secured for 400+ institutional customers, $10B+ staked on its validators, 250K+ managed nodes. Notable clients: Goldman Sachs, JP Morgan, Citi, Circle, MetaMask Institutional.
